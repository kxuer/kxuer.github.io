# 第三方库

1. base64编码：使用js-base64库
```js
import { Base64 } from 'js-base64'
var metadata = {...}
var headerValue = Object.entries(metadata).map(([key, value]) => `${key} ${Base64.encode(String(value))}`).join(',')
```
